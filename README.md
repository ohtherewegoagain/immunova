# Immunova AI
## Overview
Immunova AI is an **open-source AI-driven tool** for analyzing tumor-infiltrating lymphocytes (TILs) using **histopathological images** and **mRNA sequencing data**.

## Features
- **TIL Classification**: Predicts TIL-positive or TIL-negative cases using deep learning.
- **Histopathology & Genomics Integration**: Uses CNNs for imaging and Transformers for sequencing data.
- **Web-based Interface**: Simple UI to input data and receive instant predictions.
- **Open-Source & Extensible**: Community-driven development with modular AI components.

## Installation & Setup
### 1. Clone the Repository
```bash
git clone https://github.com/YOUR-USERNAME/Immunova-AI.git
cd Immunova-AI
```
### 2. Set Up the Frontend
Open `frontend/index.html` in a browser to test the UI.

### 3. Set Up the Backend
```bash
cd backend
pip install -r requirements.txt
python api.py
```
